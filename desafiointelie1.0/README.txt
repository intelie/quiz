Projeto de desafio proposto pela Empresa Intelie

CRIAÇÃO DO ARQUIVO EXECUTÁVEL

Para a criação do arquivo executável desta aplicação siga os seguintes passos:

1- Faça o download e a configuração da JDK(Java Development Kit) segundo as recomendações do site: http://www.oracle.com/technetwork/java/javase/install-142943.html OBS.: A JDK é necessária para os processos de compilação e execução do aplicativo

2- Faça o download e a configuração do Ant Apache segundo as recomendações do site: http://ant.apache.org/manual/install.html OBS.: O Ant Apache é necessário para a geração do executável.

3- Faça o download projeto no formato ".zip" e faça a descompactação em um local de sua preferência.

4- Acesse o diretório raiz do projeto ("desafiointelie1.0") através do terminal.

5- Digite a instrução "ant" para criar o arquivo executável "desafiointelie.jar".

6- Execute a aplicação com a instrução: java -jar desafiointelie.jar
